# **App Name**: BeClean Pro

## Core Features:

- Landing Page: Informational landing page with clear value proposition and CTAs.
- Service Pages: Dedicated pages for each cleaning service, optimized for SEO.
- Cost Calculator: Interactive calculator for estimating cleaning service costs.
- Price Display: Display of price tables editable from the CMS.
- Contact Forms: Forms for inquiries and service requests with email and Telegram notifications.
- SEO Optimization: Implementation of SEO best practices and analytics integration.

## Style Guidelines:

- Primary color: Clean and professional light blue (#ADD8E6), evoking trust and hygiene.
- Background color: Very light desaturated blue (#F0F8FF), almost white, providing a clean backdrop.
- Accent color: A slightly darker, more saturated blue (#87CEEB) used for call-to-action buttons and interactive elements.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, lending a modern and neutral appearance. Inter is suitable for both headlines and body text.
- Simple, professional icons representing cleaning services and features.
- Minimalist layout with clear sections and ample whitespace for readability.
- Subtle transitions and animations to enhance user experience without being distracting.
